# Elemental - GIMP 3.0 Theme

Elemental is a theme designed for GIMP 3.0

_Note: This theme has been tested on Ubuntu 25.04 with a 1920x1080 resolution screen.
      If you encounter any issues or have suggestions, please open an issue on the [GitHub repository](https://github.com/rohith-ck/elemental-gimp-theme)._
